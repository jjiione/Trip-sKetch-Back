# Trips Ketch - Back-End 



##  📑 프로젝트 개요

### - 프로젝트 소개
<aside>
💡 지역 경기 침체에 도움을 주고자 지역 관광 활성화를 위한 지역 관광 소개 페이지
</aside>
       
      
    

## 🛠 구현 기능

- 필수  
#1. 지역별 / 유형별 여행지 검색  
#2. 게시판 기능    
#3. 로그인 및 회원 관리 기능(마이페이지)

- 추가 및 심화  
#1. 여행 경로 설정 및 변경   
#2. 경로 저장 및 최근 경로 불러오기   
#3. 상세 정보 제공   
#4. 여행지 별 리뷰 제공
